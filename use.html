How to use Team Maker

1. Name a title for your Team!

Write it down and send it! if you want for changes, just type it on and resubmit!.

2. Toggle button

a). Divided by Group Input

For example: There are 30 items and the input is 3. Then it will make 30 items into 3 groups

b). Divided by Item Input

For example: There are 10 items and the input is 2. Then it will make 10 items divided by 2. So the total Group is 5.

maximal input value number of groups or number of people per group is the number of candidate rows you're input

3. Add candidates

Add one item name to each new line, for example:
name 1
name 2
name 3
name 4

4. Generate

To create and view the output, click random. You can re-randomise it by clicking GENERATE.

5. Edit

You can edit, add, delete members as well as delete groups of each group.

6. Save Result

You can save the results by clicking the CSV button.
